Twitter-Filter App

The Twitter Filter App is a quick app built to use the Twitter API and filtering the data based on certain criteria such as date tweeted, words contained, and twitter handle.

Visit Twitter Filter App

Features

Twitter-Filter has the following features:

  • Scrape Twitter Data with their API
  • Filtering via words
  • Filtering via date range

Technology Stack

The Twitter Filter App was built with HTML, CSS, Javascript, and JQuery on the front-end.
On the back-end the Twitter Filter App uses Python to integrate and pull data via the Twitter API.

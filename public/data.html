Data Science Projects

My data science projects include a bioinformatics research project and a social media advertising data project.

For my bioinformatics research project I created a pipeline that sequenced genomic human data and iteratively corrected the data for bias and noise. After the correction, I visualized the 3D architecture of the entire genome to determine methods of cancer detection based on nuclear architecture.

For my social media advertising project, I created algorithms that automatically queried Facebook for statistics on the social ads. After the data is collected, I created algorithms to optimize engagement and insights on social ads such as increasing ad duration during peak engagements. The data is then stored for further analysis and optimzation by the 4C Insights data science team.

Features

PITCH(Bioinformatics Project) has the following features:

  • Reading & analyzing 5GB+ of data
  • Statistically filtering & correcting data
  • Visual generation of heatmaps and circos plots
  • Human Genome Analysis between 23 chromosomes
  • Analysis of intra-chromosomal interactions

4C Insights Data Project has the following features:

  • Facebook Graph API
  • Correction & filter of data
  • Visual generation of data after correction
